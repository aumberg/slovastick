Slovastick - web-based DOM manipulator. It work with special element attributes "s-", who run actions for him, after write XPath in console or pressed keyboard shortcuts. You can automate web-surfing, navigate and edit HTML or XML document.

WARNING! Currently that program in DEEP-DEEP DEVELOPING stage. Please note. Currently work in Chrome, Firefox and Opera.


  • Get started! Lets Go!

    1. In body of every page sitting element who named "slovastick", he has its own child elements, for example "vk" for "vk.com" site, you can call them by write XPath or use keyboard shortcuts
    2. For example type in slovastick console //*[@id='getstarted']/ol/li[2] for focus on this text
    3. Slovastick can work like extension for Chrome, or like bookmarklet with manual loading in site
    4. Web developers can embed slovastick.js script for their sites

  • Shortcuts

    1. "Up + Down", or "Shift + Control" with pause - focus on slovastick console
    2. "Shift + Control + mouse move" - determine element's XPath, under mouse cursor
    3. "Arrow Up", or "Shift" - move to preceding element
    4. "Arrow Down", or "Control" - move to following element
    5. "Arrow Left", or "Shift" with pause - move to ancestor element
    6. "Arrow Right", or "Control" with pause - move to child element
