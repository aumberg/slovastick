Slovastick - web-based DOM manipulator. It work with special element attributes "s-", who run actions for him, after write XPath in console or pressed keyboard shortcuts. You can automate web-surfing, navigate and edit HTML or XML document.

That program can work like extension for Chrome , embed to website or one-page calling slovastick-bookmark for Chrome and Firefox

  • Get started! Lets Go!

    1. In body of every page sitting element who named "slovastick", he has its own child elements, for example "vk" for "vk.com" site, you can call them by write XPath or use keyboard shortcuts
    2. For example type in slovastick console command "go //*[@id='getstarted']/ol/li[2]" , and you will focused on this text
    3. Slovastick can work like extension for Chrome, or like bookmarklet with manual loading in site, where it possible
    4. Web developers can embed slovastick.js script for their sites

  • keyboard shortcuts

    1. "Up + Down", or "Shift + Control" with pause - focus on slovastick console
    2. "Arrow Up", or "Shift" - move to preceding element
    3. "Arrow Down", or "Control" - move to following element
    4. "Arrow Left", or "Shift" with pause - move to ancestor element
    5. "Arrow Right", or "Control" with pause - move to child element
